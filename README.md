# WebOS

### Introduction
#### WebOS is an interactive, browser-based operating system built specifically for a "You Ship, We Ship" (YSWS) Hack Club program. Its idea is to ship a website that is themed like an operating system to get disks. These disks are used in the market to buy different things (raspberry pi, flash drives, etc.)
---

### Functions & Features
* **Interactive Desktop Environment:** A fully functional GUI featuring draggable windows, maximize/minimize capabilities, and more features.
* **Immersive Boot & Login Sequence:** Welcomes users with a terminal-style boot screen featuring an animated Matrix effect, followed by a login screen with floating particles.
* **Dynamic Taskbar:** Includes a live, updating clock, a start menu icon, and window indicators that appear and disappear as you open and close different applications.
* **Built-in Applications:**
    * **Projects:** A dedicated space for users to add and track their projects.
    * **Market:** A store to buy different available hardware and rewards that can be purchased using earned "disks".
    * **About & FAQ:** windows explaining the program.
* **Wallpaper Customization:** Users can personalize their desktop by choosing from minimal CSS-gradient presets or by using a drag-and-drop file uploader to set custom image wallpapers. It can be found when clicking the icon in the taskbar.
