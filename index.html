FAQ

What is WebOS?

WebOS is an ysws hackclub program where you build a website that mimics an OS to get disks that can be spent in the shop to get interesting things.

Who can apply?

Any HackClub member aged between 13-18. You don't need to be in a physical club, just be part of the HackClub community.


How do I get started?

Start by creating a new project (in the projects folder). And start making your website, log your work, connect your hackatime with the project, and ship it for review when you are ready.


What is HackClub?

Hack Club is a 501(c)(3) nonprofit and network of 60k+ technical high schoolers. We believe you learn best by building so we're creating community and providing grants so you can make awesome projects. We run programs like these year around.
